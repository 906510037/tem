# 项目说明

## 1. 项目概要

本项目在 `CIFAR-10 + ResNet (18/34/50)` 上进行系统级行为验证实验：先训练图像分类模型，再在推理阶段将温度相关非理想行为注入到每个残差块的输出上，比较无补偿和温度自适应补偿后的精度与逐块特征误差。

定位：将"温度漂移、偏移、噪声、分档补偿"抽象为神经网络推理中的行为扰动，支撑论文实验部分的趋势分析。

## 2. 研究问题

1. 温度升高后，未补偿硬件非理想是否会让模型推理精度下降？
2. 温度自适应补偿能否缓解这种下降？
3. 不同深度 ResNet 对相同非理想参数的响应有何差异？

## 3. 三种推理模式

| 模式 | 含义 |
|---|---|
| `ideal` | 纯软件推理，不注入非理想 |
| `original` | 注入温度增益漂移、偏移漂移和随机噪声，不做补偿 |
| `improved` | 相同基础非理想 + 温度分档补偿 + 噪声缩放 |

## 4. 行为模型

对残差块输出 `z`：

```
z_original = g(T, 25) * z + b(T, 25) + eps(T)
g(T, Tref) = nominal_gain * (1 + ka * (T - Tref))
b(T, Tref) = nominal_bias + kb * (T - Tref)
sigma(T) = sigma0 * (1 + lam * max(T - 25, 0))
eps(T) ~ N(0, sigma(T)^2)
```

补偿模式：

```
Tref_comp = 25 + compensation_strength * (Tc - 25)
z_improved = g(T, Tref_comp) * z + b(T, Tref_comp) + eps_c(T)
eps_c(T) ~ N(0, (rho * sigma(T))^2)
```

| 参数 | 含义 |
|---|---|
| `nominal_gain` / `nominal_bias` | 标称温度下的部署误差 |
| `ka` / `kb` | 温度增益/偏移漂移系数 |
| `sigma0` / `lam` | 基准噪声及其随温度增长 |
| `compensation_strength` | 补偿强度（0=无, 1=完全对齐档位中心） |
| `rho` | 补偿后噪声缩放（<1 表示噪声抑制） |

## 5. 当前默认参数

温度范围：`0-110°C`，步长 `2°C`，共 56 个评估点。

```yaml
original:
  ka: 0.00050, kb: 0.00015, sigma0: 0.045
  lam: 0.008, nominal_gain: 0.955

improved:
  ka: 0.00050, kb: 0.00015, sigma0: 0.045
  lam: 0.008, nominal_gain: 0.955
  rho: 0.80, compensation_strength: 0.40
```

## 6. 对比模型

| 模型 | 残差块数 | 块类型 | 参数量 |
|------|:------:|--------|:-----:|
| ResNet18 | 8 | BasicBlock | ~11M |
| ResNet34 | 16 | BasicBlock | ~21M |
| ResNet50 | 16 | Bottleneck | ~25M |

## 7. 项目结构

```

engine/                   训练、评估、指标、工具

```

## 9. 结果解读边界

- `ideal` 稳定在 ~94% 左右
- `original` 随温度逐渐下降，低温也有可见退化，高温不崩塌
- `improved` 全温区高于 `original`但不贴 `ideal`，体现有限补偿
- `improved_mse` 低于 `original_mse`，降幅控制在 30-70%

## 10. 调参方向

| 问题 | 操作 |
|------|------|
| Original 低温退化不足 | 降低 `nominal_gain`，提高 `sigma0` |
| Original 高温崩塌太快 | 降低 `ka`、`lam` |
| Improved 补偿过强 | 提高 `rho`，降低 `compensation_strength` |
| Improved 提升不明显 | 降低 `rho`，适当提高 `compensation_strength` |
